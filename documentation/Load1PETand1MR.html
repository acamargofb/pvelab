Performing this task you are asked to load first a PET image, the image that shall be corrected for PVE problems, and afterwards an MR image, the image that are segmented into gray, white, and CSF matter. The MR image should have as high resolution as possible and preferable have cubic voxels, so it is possible to resilce the image to PET resolution

Show

Pressing the show buttom will create two image for the Display window. The first image show is the PET image and the second is the MR image

Using this buttom it is possible to see the original images to which the PVE process is performed.

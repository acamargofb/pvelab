browse2d

The browse2d program visualises (in principle) any number of registered datasets on a series of simple, 2d images. The disadvantage of the program is that the images must all be resliced and written to disk before the program can run. The program has a main window (Figure 0.2.3) and a series of client windows (Figure 0.2.3). Clicking a point on the shown image in a client window or changing the slider of a client window, will update all other client windows.

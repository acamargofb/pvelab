browse3d

The browse3d program is a 3d analogue of the browse2d program, but is much more memory consuming, since all full volume datasets must be loaded. The browse3d program visualises (in principle) any number of registered datasets on a series of 3-pane windows. The disadvantage of the program is that the images must all be resliced and written to disk before the program can run. The program has a main window (Figure 0.2.4) and a series of client windows (Figure 0.2.4). Clicking a point on the shown image in a client window or changing the slider of a client window, will update all other client windows.

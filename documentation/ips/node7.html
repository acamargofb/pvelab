Exiting

Once registration is achieved, a point definition file should be written to disk, to save the selected points for later reference. If point definitions have not been saved on exit, the program will prompt the user for this action. Point definition files are known to IPS only, MARS does not have a feature for interpreting these files. Exiting the IPS will return the user to the main interface of MARS, where registration parameters can be saved.
